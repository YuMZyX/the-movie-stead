The Movie Stead

STACK: React JS / Node.js / PostgreSQL

* Uses TMDB API
* ROLES: Visitor / User / Moderator


Visitors can:
- Search/filter movies by different category
- Sign up / Sign in

| Movies | 						                    | Login |
							                             -Register
User can:
- Review movies
- Browse/sort own reviewed movies
- Modify own reviews (only 1 review per movie)
- Delete own reviews
- Add movies to watchlist
- Browse/sort own watchlist
- Remove movies from own watchlist

| Movies | Watchlist | My reviews |			  | Log out |
 -Review

Moderator can:
- Do everything that User can
- Browse/sort users
- Disable users
- View each users reviews
- Modify or delete unappropriate reviews

| Movies | Watchlist | My reviews | Users | Log out |
 -Review	                         -Reviews
